# CISC 101 – Paper Summarizer Project (Group 85)

This repository contains our modular AI prompt system for the Research Paper Summarizer.

## Repository Structure

- `system_prompt.md` — Main system prompt defining behavior, rules, tone, and output format.
- `modules/` — Internal reference framework with all modules.
    - `01_intake_setup.md`
    - `02_section_loop.md`
    - `03_guardrails.md`
    - `04_render_refine.md`
    - `05_student_module_a.md`
    - `06_student_module_b.md`

## Purpose
This project implements a modular LLM system that:
- summarizes academic research papers,
- supports strict evidence mode,
- detects missing/short sections,
- handles multiple summary levels,
- generates expert & lay summaries,
- and includes student-created modules that extend functionality.
